# 6–8 minute group demonstration
- **Member 1 (1 min):** Problem, features, package/resource architecture.
- **Member 2 (2 min):** Sign-up/login validation; show users schema and password hashing.
- **Member 3 (2 min):** Add/list/delete notes; explain `BaseAdapter`, recycled row, and FAB listener.
- **Member 4 (1.5 min):** Bottom navigation, account details, SharedPreferences, three theme modes.
- **Member 5 (1 min):** Privacy statement, testing, security limitations, future improvements.

Every member should be ready to explain one SQL statement, one listener, one resource XML file, and one validation path.
